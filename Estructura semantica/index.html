Estructura semantica

Anteriormente en HTML4 habia pocas etiquetas semnaticas. Para los casos en los que no habia una etiqueta esecifica se usaban dos etiquetas genericas:



			div  (block) 
			span (inline)

En HTML5 se introducen nuevas etiquetas semanticas para elementos block e inline. Estas pueden aparecer varias veces menos el main, el main es unico

Para los elementos block: 

			header 	(Encabezados) 													
			nav 	(navegacion, para poner menus)						
			footer  (pie de pagina)										
			article (contenido independiente)							
			aside   (contenido relacionado pero secundario)				
			section (secciones o divisiones de un contenido)			
			main 	(Para mostrar contenido principal, debe ser unico)
